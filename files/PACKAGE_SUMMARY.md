## üìä **DATABASE OVERVIEW**

**Core Tables:**
1. `gyms` - Multi-tenant gym information
2. `gym_users` - Staff with roles (+ `preferred_language`)
3. `members` - Gym members (+ `preferred_language`)
4. `membership_plans` - Plans (+ `name_translations`)
5. `subscriptions` - Active memberships
6. `payments` - Transactions (+ `due_date` for calendar)
7. `classes` - Class types (+ `name_translations`)
8. `class_schedules` - Scheduled sessions
9. `bookings` - Member class bookings
10. `check_ins` - Attendance logs
11. `notifications` - Communications (+ `recipient_language`)

**Security:** Row Level Security (RLS) on all tables
